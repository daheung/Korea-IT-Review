day30(js 2일차)
함수 -> 객체 -> 배열 -> map/filter -> 자바스크립트 엔진 구조 -> DOM

1. 함수(Function)
   여러줄의 코드를 하나로 묶은 실행 단위
   자바스크립트의 함수는 일급 객체이다(일급객체 : 값으로 사용할 수 있는 것을 의미)
   즉, 함수는 객체 타입의 값이며 변수에 저장하거나 매개변수로 전달/반환하는 것이 가능하다

1) 함수 명명 규칙
   함수이름 = 동사 + 명확한 역할
   이름만 봐도 무슨일을 하는지 알 수 있어야한다
   ex) showMessage()    getAge()   setName()

2) 함수 설계 규칙
   함수는 하나의 동작만
   이름에 나온 일만 해야 함
   서로 다른 일을 하는 함수는 분리

- 잘못된 함수 설계
   function login(){
      //로그인 처리
      //로그파일 저장
      //화면이동   
   }

- 좋은 설계
   login()
   seveLog()
   movePage()


3) 함수 종류
(1) 일반함수
   function 식별자(매개변수1, 매개변수2){
      //코드작성
      return 리턴값;
   }

- 용어
   function   함수 선언 표시
   식별자      함수의 기능을 의미할 수 있는 동사로 작성
   parameter   매개변수(인자), 여러개 있을 때는 콤마로 분리하고 자료형을 따로 작성하지 않는다
   return      함수 실행 종료 후 리턴값을 반환한다

- 함수 호출 방식
   매개변수   리턴값   호출방법
   x   x   식별자();
   o   x   식별자(인수);
   x   o   출력(식별자());
   o   o   출력(식별자(인수));

- 매개변수와 인수
   매개변수(인자, 파라미터, parameter) : 매개변수(함수 선언부)
   인수(아규먼트, argument) : 매개변수에 들어가는 값(함수 사용부, 호출부)
   
   function (a, b){   // a, b 매개변수
      retrun a + b;
   } add(10, 20); //10, 20 인수

   디폴트 매개변수 : 값이 안들어왔을 때 기본값
      기본값은 맨 뒤에서부터 설정한다(매개변수의 마지막)   

   function numberPrint(num1, num2, num3=10){
      console.log(num1, num2, num3);
   }

   numberPrint(10, 20, 30);

   가변 매개변수(rest parameter) : 인수가 몇 개 올지 모를 때
      배열로 만들어진다

   function numPrint(...number){
      출력(number);
      for(int i = 0; i < number; i++){
         출력(i);
      }
   }

+) 동적바인딩
   자바스크립트는 실행 시점에 타입이 결정
   
(2) 익명함수 ★★★★★
   이름이 없는 함수이며 보통 변수에 저장해서 사용한다
   
   let 변수명 = function (매개변수){
         실행할 코드;
         return 반환값;
      };

(3) 콜백함수 ★★★★★
   함수의 매개변수로 전달되는 함수를 의미한다(다른 함수의 매개변수로 전달되어 실행되는 함수를 의미한다)
   모듈화를 시키기 위해 원하는 곳에서 실행할 때 사용한다(재사용성)
   비동기작업, 재사용성, 모듈화를 위해 사용한다

   button.addEvenetListener("click", function() { 
               console.log("클릭됨");
            });
   
- 콜백지옥 : 콜백함수의 단점
   콜백을 과도하게 중첩하면 코드가 복잡해지고 가독성이 떨어지는 콜백 지옥이 발생할 수 있다
   
   a(function() {b (function() {c (function() { d (function(){
      ...
   }); }); }); });

- 콜백함수의 대안 : Promise와 async / await
   코드의 가독성을 높이고 비동기작업을 체계적으로 처리할 수 있다

(4) 즉시 실행함수
   한번만 사용하는 함수는 즉시 실행함수로 만들어 사용한다
   즉시 실행함수는 만듦과 동시에 실행이 된다
   재사용할 목적이 없으므로 익명함수로 만든다

   (function (매개변수){
      실행할코드;
   })(매개변수로 넘겨줄 값);

   정의하자마자 실행, 전역 오염 방지, 테스트코드에서 자주 사용된다

(5) 화살표함수 ★★★★★
   익명함수의 축약형
   콜백함수에서 압도적으로 많이 사용되며 this 바인딩이 다르다!

   () => {}

(6) 재귀함수
   자기 자신을 호출하는 함수

2. 전역 객체
    GlobalThis
      웹 브라우저 환경에서는 windows
      Node.js 환경에서는 global

3. 객체
1) javascript 객체
   자바스크립트는 원시 타입값을 제외한 모든 값이 객체
   객체는 프로퍼티(property)와 메소드로 구성된다
   프로퍼티는 키와 값으로 구성되며 값의 위치에 따라 함수가 올 수 있다
   프로퍼티의 값이 함수인 경우 일반 함수와 구분하기 위해 메소드라고 부른다

2) 객체 생성
   클래스가 없어도 다양한 방법으로 객체를 만들 수 있다
   객체 리터럴을 이용한 생성이고 객체 리터럴은 {}을 이용하여 표현한다

   let test = {
      num1 : 10,
      num2 : 20,
      name : "짱구",
      sum : function(){
         return this.num1 + this.num2;
      }
   };

3) 객체 사용
   객체의 프로퍼티에 접근하는 방법은 2가지가 있다
   - .을 이용한다
      console.log(test.name);   //짱구
   - []을 이용한다(key를 반드시 ''로 감싸야한다)
      console.log(test['name']); //짱구

4) 프로퍼티의 생성과 삭제
   객체를 생성하고 이후에 객체 내부에 프로퍼티를 생성하거나 삭제하는 것이 가능하다
   - 생성
      객체에 존재하지 않는 키에 값을 저장한다
      test.num3 = 3.5;
   - 삭제
      delete를 사용한다
      delete test.num3;

4. 생성자 함수와 클래스
1) 생성자 함수
   함수 선언을 통해 생성자를 정의한다
   생성자 함수 내부에서 this 키워드를 사용하여 객체의 프로퍼티를 초기화한다
   new 키워드를 이용하여 생성자 함수를 호출하여 객체를 생성한다

2) 클래스
   class 키워드를 사용하여 클래스를 정의한다
   클래스 내부에 constrctor 메소드를 정의하여 객체의 프로퍼티를 초기화한다
   new 키워드를 이용하여 클래스를 호출하여 객체를 생성한다
